#  重写 `myl7` 项目的 benchmark，实现对 bin、bout 及测试规模等参数的可控配置。

实现myl7和orca的dpf/dcf性能对比

orca测试参数：
./dpf 64 10000000
./dpf 64 1 10000000

myl7似乎无法这样设定参数，难以横向比较

## 2026-04-04 按新要求收口

用户新要求：

- 只保留 `DPF bin=64`
- 只保留 `DCF bin=64, bout=1`
- `N` 可以做梯度测试
- 其余参数记录全部删除

因此下面只保留这一组口径下的实现、命令和测试结果。

### 当前目录新增文件

- `run_myl7_bench.py`
  负责生成、编译并运行当前目录下的专用 benchmark。
- `myl7_gpu_bench_template.cu.in`
  benchmark 模板。
- `.gitignore`
  忽略当前目录下的 `.build/` 缓存产物。

### 当前口径下的使用方式

DPF：

```bash
python3 260404重构myl7/run_myl7_bench.py dpf 64 N --chunk CHUNK_N
```

DCF：

```bash
python3 260404重构myl7/run_myl7_bench.py dcf 64 1 N --chunk CHUNK_N
```

说明：

- 不传 `--chunk` 时，脚本会按当前显存自动估算安全 `chunk_n`
- 生成出的源码和二进制会缓存在 `260404重构myl7/.build/`
- benchmark 会做结果重构校验，避免只快不对
